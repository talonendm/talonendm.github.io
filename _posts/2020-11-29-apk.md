# apk
"Android Package (APK) is the package file format used by the Android operating system, and a number of other Android-based operating systems for distribution and installation of mobile apps, mobile games and middleware." [wikipedia](https://en.wikipedia.org/wiki/Android_application_package)

{: .box-warning}
**Warning:** Never download apk-files from untrusted pages!

So, if you know and trust me, go ahead. Files with .apk and .exe extension are mainly meant for myself. In year 2040 I still can easily install my own worm game to my laptop!
